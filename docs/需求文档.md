# MindEcho 智愈心海 - 需求文档

## 1. 项目概述

### 1.1 项目名称
**MindEcho - 智愈心海**

### 1.2 项目定位
一款基于 JavaFX 的桌面端情绪健康管理应用，致力于为用户提供安全、温暖的情绪倾诉与疗愈空间。通过 AI 对话、情绪胶囊、心情报告、文章推荐等多元化功能，帮助用户认知、接纳和管理自己的情绪。

### 1.3 目标用户
- 有情绪倾诉需求的年轻人
- 希望了解和管理自身情绪的人群
- 寻求心理疗愈和自我成长的用户

### 1.4 技术栈
| 类别 | 技术 |
|------|------|
| 开发框架 | JavaFX 21 |
| 项目管理 | Maven |
| 数据存储 | SQLite |
| 架构模式 | MVC |
| 主题样式 | CSS（深色/浅色双主题） |
| AI 集成 | OpenAI API |
| 第三方 API | ZenQuotes、Hitokoto、ApiZero |
| 音频播放 | JavaFX MediaPlayer |
| 网络请求 | OkHttp 4.x |
| JSON 解析 | Gson 2.x |
| 加密 | AES-256-GCM |

## 3. 功能详细需求

### 3.1 情绪倾诉模块（主页面）

#### 3.1.1 功能描述
用户可以在主页面自由倾诉心情，AI 会给出温暖回应，并自动分析情绪类型和压力等级。

#### 3.1.2 功能需求
- **文本输入**：支持多行文本输入，可输入任意长度的心情描述
- **AI 对话**：
  - 两种 AI 回应风格：温柔型 / 犀利型
  - 异步生成回应，避免界面卡顿
  - API 未配置时使用本地备用语录
- **情绪分析**：
  - 自动识别情绪类型（开心/愤怒/焦虑/悲伤/平静）
  - 压力等级评估（1-5级）
  - 能量值检测
- **心情天气**：
  - 根据当日情绪自动计算心情天气状态
  - 配合环境音效营造氛围
- **数据存储**：
  - 对话记录加密存储
  - 按日期索引查询

#### 3.1.3 交互要求
- 输入框支持回车发送
- AI 回复时有加载状态提示
- 情绪标签用彩色徽章展示
- 对话气泡有淡入动画

---

### 3.2 AI 情绪胶囊模块

#### 3.2.1 功能描述
用户可以将当下的情绪封存进「时间胶囊」，经过设定的时间后才能解锁查看，并获得 AI 生成的成长回顾。

#### 3.2.2 功能需求
- **胶囊封存**：
  - 输入想要封存的情绪内容
  - 自动分析情绪标签和压力等级
  - 选择解锁时间（3天/7天/30天）
  - 胶囊存入「胶囊墙」
- **胶囊墙展示**：
  - 网格布局展示所有胶囊
  - 显示情绪图标、封存日期、解锁倒计时
  - 已解锁胶囊有特殊视觉效果
  - 支持按状态筛选（全部/未解锁/已解锁）
- **胶囊解锁**：
  - 到达设定时间后方可打开
  - 打开时有翻转动画效果
  - 显示当时封存的内容和情绪
  - AI 生成「成长回顾」寄语
- **每日限额**：
  - 每天最多封存 3 个胶囊
  - 显示当日剩余额度

#### 3.2.3 数据结构
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 胶囊唯一标识 |
| content | String | 封存内容 |
| emotion | String | 情绪标签 |
| stressLevel | int | 压力等级 1-5 |
| createdAt | LocalDateTime | 封存时间 |
| unlockAt | LocalDateTime | 解锁时间 |
| opened | boolean 是否已解锁 |
| growthReview | String | AI 成长回顾 |

### 3.3 心情报告模块

#### 3.3.1 功能描述
基于用户的倾诉记录，生成可视化的情绪统计报告，帮助用户了解自己的情绪模式。

#### 3.3.2 功能需求
- **情绪统计**：
  - 各情绪类型占比饼图
  - 情绪分布柱状图
  - 压力等级趋势图
- **时间维度**：
  - 本周报告
  - 本月报告
  - 自定义时间段
- **关键指标**：
  - 倾诉总次数
  - 情绪正能量占比
  - 平均压力值
  - 最高频情绪
- **月度报告**：
  - 月度情绪总结
  - 成长建议
  - 对比上月变化

#### 3.3.3 可视化要求
- 使用 JavaFX 原生图表组件
- 配色与整体主题协调
- 支持数据悬停查看详情

### 3.4 情绪文章推荐模块

#### 3.4.1 功能描述
提供情绪释放相关的文章推荐，帮助用户学习情绪管理技巧和心理学知识。支持 AI 心情识别、个性化文章推荐和环境音效联动。

#### 3.4.2 功能需求
- **AI 心情识别**：
  - 用户输入心情描述文本
  - AI 自动分析情绪类型（愤怒/焦虑/悲伤/平静）
  - AI 评估压力等级（1-5级）
  - 识别结果自动映射到文章主题标签
  - 自动高亮对应主题标签
- **心情识别与音效联动**：
  - 识别情绪后自动切换环境音效
  - 情绪→音效映射：
    - 愤怒 → ⚡️ 雷雨（低沉鼓点+压迫性风声）
    - 焦虑/压力（高压力）→ ⚡️ 雷雨
    - 焦虑/压力（低压力）→ ☁️ 阴天（低频心率+风声）
    - 悲伤 → 🌧️ 小雨（雨声+空间混响）
    - 平静/快乐 → 🌤️ 晴天（自然白噪音+轻风）
  - 音效淡入淡出过渡（2秒）
  - 音效开关联动（设置中关闭音效时不切换）
- **文章分类浏览**：
  - 按主题标签筛选（情绪管理、压力管理、悲伤疗愈、焦虑管理、放松技巧、自我成长、心态调整、自我探索）
  - 「全部」标签查看所有文章
- **文章卡片展示**：
  - 情绪图标
  - 文章标题
  - 内容摘要
  - 主题标签
  - 「阅读全文」按钮
- **文章详情**：
  - 弹窗展示完整内容
  - 支持滚动阅读
  - 显示作者和来源
  - 打开时有缩放淡入动画
- **AI 生成文章**：
  - 选择情绪主题生成定制文章
  - 支持 7 种预设情绪（焦虑/压力/悲伤/愤怒/快乐/平静/迷茫）
  - 支持自定义情绪输入
  - 生成过程有加载提示
- **网络文章导入**：
  - 一键从第三方 API 导入精选文章
  - 集成多个开放 API 源：
    - ZenQuotes（英文箴言）
    - Hitokoto 一言（中文多分类）
    - ApiZero 语录（中文经典语录）
  - 自动将语录扩展为完整文章格式
  - 标注文章来源平台
  - 网络异常时使用本地备用内容
- **文章来源标识**：
  - 预置文章：心灵驿站
  - AI 生成：AI 心灵导师
  - 网络导入：显示来源平台

## 4. 非功能需求

### 4.1 性能要求
- 应用启动时间 < 3 秒
- 页面切换响应时间 < 500ms
- AI 对话异步处理，不阻塞界面
- 数据库查询 < 200ms

### 4.2 安全要求
- 对话内容加密存储
- API Key 安全保存，不明文输出日志
- 本地数据隔离，不上传云端

### 4.3 兼容性要求
- 支持 Windows 10/11
- 支持 macOS 10.15+
- 支持 Linux（主流发行版）
- 最低分辨率：860×640

### 4.4 可用性要求
- 界面简洁，操作直观
- 新用户无需学习即可上手
- 所有功能均可在 3 次点击内到达
- 错误提示友好，给出解决建议

### 4.5 可扩展性要求
- 模块化设计，便于新增功能
- 服务层接口化，可替换实现
- 主题系统支持新增主题
- 数据库迁移方案预留

## 7. 第三方服务集成

### 7.1 OpenAI API
- **用途**：AI 对话、情绪分析、文章生成、成长回顾
- **接口**：Chat Completions API
- **模型**：gpt-4o-mini（默认）
- **配置项**：API Key、Base URL、模型、最大 Token

### 7.2 第三方文章 API

#### 7.2.1 ZenQuotes
- **地址**：https://zenquotes.io/api/random
- **特点**：免费、无需密钥、英文箴言
- **用途**：网络文章导入源

#### 7.2.2 Hitokoto 一言
- **地址**：https://v1.hitokoto.cn/
- **特点**：免费、中文、多分类
- **分类**：文学、动漫、哲学、影视等
- **用途**：网络文章导入源

#### 7.2.3 ApiZero 语录
- **地址**：https://apizero.cn/marketplace/hitokoto
- **特点**：免费、中文经典语录
- **用途**：网络文章导入源
